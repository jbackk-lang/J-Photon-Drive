# Powiązanie z modułem matematycznym

Model operatora J korzysta z matematycznej struktury walidacji operatorów
z projektu `math-validator`.

## Laplasjan

Δf = 0

Warunek ten określa:

- dopuszczalne mody pola,
- położenie punktu J,
- stabilność rezonansu.

## Operatory

- B1 — zapis stanu
- J — skręt + rezonans
- B2 — odczyt / przywrócenie

Walidacja operatorów odbywa się poprzez:

- sprawdzanie unitarności,
- sprawdzanie odwracalności,
- zgodność z równaniami rezonansu.
