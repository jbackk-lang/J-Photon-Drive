# Model operatora J w eksperymencie akceleratorowym

## Założenia częstotliwości

- Częstotliwość RF: \(\nu = 500\,\text{MHz}\)
- Prędkość światła: \(c = 3 \cdot 10^8\,\text{m/s}\)
- Długość fali:
  

\[
  \lambda = \frac{c}{\nu} = 0{,}6\,\text{m}
  \]



## Geometria helisy i operator J

- Jedna strona helisy:
  

\[
  L = \frac{\lambda}{2} = 0{,}3\,\text{m}
  \]


- Obieg tam + z powrotem:
  

\[
  2L = 0{,}6\,\text{m}
  \]


- Czas obiegu:
  

\[
  T_{\text{obieg}} = \frac{2L}{c} \approx 2\,\text{ns}
  \]



Operator **J**:

- J = punkt skrętu helisy + dwie strony pola + rezonans.
- Warunek rezonansu:
  

\[
  2L = n\lambda
  \]


- J działa jako przywracacz pamięci, gdy:
  

\[
  B_2\, J\, B_1 \approx I
  \]



## Pamięć (heliosa) i koherencja

- Heliosa = stan wzbudzony helu pełniący rolę pamięci pola.
- Warunek koherencji:
  

\[
  T_{\text{koherencji}}^{\text{heliosa}} \gg T_{\text{obieg}} \approx 2\,\text{ns}
  \]



## Interpretacja fizyczna

- RF = 500 MHz ustala częstotliwość fali, na której:
  - przyspieszane są cząstki,
  - generowane są fotony,
  - działa helisa i operator J.
- Laplasjan pola wyznacza:
  - dopuszczalne mody,
  - położenie bram \(B_1, B_2\),
  - położenie punktu J (maksima pola).

Model w tej wersji traktowany jest jako koncepcja eksperymentu z podwójną bramą i pamięcią pola w akceleratorze.

# Model operatora J

Operator **J** reprezentuje punkt skrętu helisy pola w eksperymencie
akceleratorowym. Łączy dwie strony pola elektromagnetycznego oraz warunek
rezonansu wynikający z Laplasjanu.

## Parametry RF

- częstotliwość RF: 500 MHz
- długość fali:
  λ = c / ν = 0.6 m

## Geometria helisy

- jedna strona helisy: L = 0.3 m
- obieg tam + z powrotem: 2L = 0.6 m
- czas obiegu: T = 2 ns

## Warunek rezonansu

2L = n λ

## Rola operatora J

J działa jako punkt skrętu, w którym fala powrotna jest w fazie z falą
pierwotną. Umożliwia to:

- przywracanie pamięci pola,
- odwracanie działania bram B1 i B2,
- tworzenie fali stojącej w helisie.

Formalnie:

B₂ J B₁ ≈ I
